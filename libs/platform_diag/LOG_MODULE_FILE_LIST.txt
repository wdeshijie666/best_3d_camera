libs/platform_diag/CMakeLists.txt

## 说明
- 根工程 CMakeLists.txt 中有 spdlog 的查找与 `camera3d_spdlog_headers` 的 INTERFACE 回退；若他组独立成库，可只使用上方文件 + 自写 `find_package(spdlog CONFIG REQUIRED)` 或头文件包。
- 不列出根 CMakeLists 本身，避免误拷整仓；他组以本文档「依赖」与 README_LOG_MODULE.md 为准集成。

## 同目录说明文档
- README_LOG_MODULE.md：日志、DiagConfig、集成步骤。
- README_CRASH_HANDLER.md：InstallCrashHandlers、Windows MiniDump、terminate 与调用顺序。
- EXAMPLE_CMAKE_SPDLOG.md：例如 D:/libs/spdlog 下的 CMake 与 main 最小示例（日志+crash_handler）。
- examples/minimal_app/：可独立 cmake 的示例工程（CMakeLists.txt + main.cpp + README.txt）。
